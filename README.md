# Moataz Mohammady --- Resume Website

A personal resume website built with **GatsbyJS** and styled using
**Chakra UI**.\
This project highlights a fast, modern, and accessible frontend setup.

## 🚀 Tech Stack

### Core Packages

-   [GatsbyJS](https://www.gatsbyjs.com/) --- Static site generator for
    blazing-fast websites\
-   [Chakra UI](https://chakra-ui.com/) --- Simple, modular, and
    accessible component library

### Gatsby Plugins

-   `gatsby-plugin-google-analytics` --- Adds Google Analytics tracking
    to your Gatsby site

## 🛠️ Installation & Setup

Install dependencies:

``` bash
yarn install
```

Run the development server:

``` bash
yarn dev
```

Build the production version:

``` bash
yarn build
```

Preview the production build:

``` bash
yarn serve
```

## 📁 Project Structure

    /src
      /components   → Reusable UI components
      /pages        → Site pages
      /theme        → Chakra UI theme overrides
      /assets       → Images, icons, fonts
    gatsby-config.js
    package.json
